Summary of ISO8601 Standard

Description

A summary and description how Rmetrics uses the ISO8601 standard.

isISO8601 test if the time/date is ISO8601 formatted.

Usage

isISO8601(x)

Arguments

x an argument to be checked.

Value

isISO8601
returns TRUE if x is ISO8601 formated, i.e. as "%Y-%m-%d %H:%M:%S", as "%Y-%m-%d", as "%Y%m%d%H%M%S", or as "%Y%m%d".

Author(s)

Diethelm Wuertz for this summary.

Examples

## SOURCE("fCalendar.1A-ISO8601Standard")

## isISO8601 -
    isISO8601(c("2007-01-01", "2007-12-31" ))
    # Don't print control attribute:
    isISO8601(c("2007-01-01", "2007-12-31" ))[[1]]
    # Nin-ISO8601 
    isISO8601("2007-Jan-01")[[1]]
    isISO8601("2007-01-01 15:00:000")[[1]]
