print.acdModel(fACD)R Documentation

Function to print ACD model

Description

Prints the coefficients of the model to the screen

Usage

print.acdModel(modelOut)

Arguments

modelOut A S4 object from ACD_Fit

Value

Returns nothing

Author(s)

Marcelo Perlin - ICMA/UK <marceloperlin@gmail.com>

See Also

plot.acdModel

Examples


data(myDur)

x=myDur
pLag=1
qLag=1
distrib='exp'

typeACD='ACD'

modelOut<-ACD_Fit(x,qLag,pLag,distrib,typeACD)
print(modelOut)
