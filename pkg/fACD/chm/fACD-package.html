fACD-package(fACD)R Documentation

Package for Estimation and Simulation of ACD Models

Description

This package can estimate and simulate ACD (Autoregressive Conditional Duration) model, with support for the ACD, log, BC and EX specifications, also including the use of the exponential and the weibull distribution
