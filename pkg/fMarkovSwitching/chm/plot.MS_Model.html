plot.MS_Model(fMarkovSwitching)R Documentation

Function to Plot the Fitted Processes of a Markov Switching Model

Description

Using as input a S4 "MS_Model" object, this function plots:
-Fitted Filtered Probabilities (t¦t)
-Fitted Smoothed Probabilities (by means of Kim's Algorithm)
-Fitted Conditional Mean (based on filtered probabilities of t¦t)
-Fitted Conditional standard Deviation (based on filtered probabilities of t¦t)

Usage

plot(MS_Model_In)

Arguments

MS_Model_In A S4 object (a output from MS_Regress_Fit)

Value

Returns Nothing. Plots the time series in a graphical device
