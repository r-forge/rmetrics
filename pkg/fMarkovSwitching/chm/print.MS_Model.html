Function to Print (output to screen) a Markov Switching Model

Description

This function will output to screen a model fitted with MS_Regress_Fit

Usage

print(MS_Model_In)

Arguments

MS_Model_In A S4 object obtained with MS_Regress_Fit

Value

Returns nothing. Prints model in screen
