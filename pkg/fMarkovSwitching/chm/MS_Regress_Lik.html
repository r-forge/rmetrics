MS_Regress_Lik(fMarkovSwitching)R Documentation

The likelihood function for a General Markov Switching Model

Description

This function calculates the log likelihood of the model and is maximized by Rdonlp2. You'll probably never have to call this function (MS_Regress_Fit does that for you..)

Usage

MS_Regress_Lik(param)

Arguments

param a vector with parameters, only. All the rest of the input arguments are passed as global variables (see rdonlp2 documentation)

Value

Returns the sum of log likelihood. It may also return the filtered series of the model (see MS_Regress_Fit code)
