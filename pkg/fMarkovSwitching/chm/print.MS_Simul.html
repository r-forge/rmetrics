print.MS_Simul(fMarkovSwitching)R Documentation

Function to Print a Markov Switching Simulation Object

Description

This function will print to screen a Markov Switching object obtained with MS_Regress_Sim

Usage

print.MS_Simul(MS_Simul_In)

Arguments

MS_Simul_In A S4 object obtained with MS_Regress_Sim

Value

Returns nothing. Prints simulated model to screen
